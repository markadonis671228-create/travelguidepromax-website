Reflection

Creating Travel Guide was both challenging and rewarding. Each member applied their HTML5 and CSS knowledge to create pages that complement one another. This project enhanced our understanding of website structure, layout design, and teamwork in web development.

Note: Some formatting and design enhancements were assisted by AI tools, particularly in refining the CSS for responsiveness, spacing, and layout styling. However, all original HTML, structure, screenshots, and written content were created by our team based on our learning and effort.
